🛡 Proteção Anti-ban

Janela de horário

Mensagens só são enviadas dentro do horário configurado. Fora dele, os jobs ficam na fila e disparam automaticamente quando a janela abre.
Exemplo: início 8, fim 20 → dispara das 8h às 19h59.

Spintax — variação automática de texto

Use {opção1|opção2|opção3} no template para variar o texto a cada mensagem enviada.
Suporta aninhamento: {Olá|{Oi|Ei}, tudo bem?}

Exemplo completo:
{Olá|Oi|Bom dia}, {nome}! 👋
{Temos uma novidade|Preparamos algo especial} para você{!| hoje!| esta semana!}

Como configurar
1 Criar projeto no Google Cloud ▼

1. Acesse console.cloud.google.com

2. Clique em Selecionar projeto → Novo projeto

3. Dê um nome (ex: "disparador") e clique em Criar

2 Ativar Google Sheets API ▼

1. No menu lateral, vá em APIs e serviços → Biblioteca

2. Busque por Google Sheets API

3. Clique em Ativar

3 Criar Service Account e baixar JSON ▼

1. Vá em APIs e serviços → Credenciais

2. Clique em Criar credenciais → Conta de serviço

3. Preencha o nome e clique em Concluído

4. Clique na conta criada → aba Chaves → Adicionar chave → JSON

5. Um arquivo JSON será baixado — cole o conteúdo abaixo

4 Compartilhar planilha com a Service Account ▼

1. Abra sua planilha no Google Sheets

2. Clique em Compartilhar (botão verde, canto superior direito)

3. Cole o e-mail da service account (termina em @...iam.gserviceaccount.com)

4. Selecione permissão Visualizador e clique em Enviar

💡 O e-mail da service account está dentro do JSON que você baixou, no campo "client_email"
5 Formato da planilha ▼

A primeira linha deve ser o cabeçalho. A coluna numero é obrigatória:

numero | nome | cidade | produto
11999998888 | Maria | São Paulo | Plano A
21988887777 | João | Rio | Plano B

As colunas extras ficam disponíveis como {variáveis} no template da campanha.

O ID está na URL da planilha: docs.google.com/spreadsheets/d/ID_AQUI/edit

🔄 Sincronização automática Sheets → Banco

Configure para importar contatos do Sheets automaticamente em intervalos regulares. O banco sempre reflete a planilha.

🔔 Webhook (notificações em tempo real)

O webhook permite que a Evolution API notifique o sistema instantaneamente quando um chip conecta ou desconecta, sem precisar de polling. Recomendado para uso em produção.
⚠ Para uso local, o webhook só funciona se a Evolution API estiver na mesma máquina. Para acesso externo, use um serviço como ngrok ou hospede em um servidor público.
